Git repository létrehozása

1. lépés

Először hozzunk létre egy mappát. Ebből fogunk repository-t csinálni.

CMD-ben mkdir mappa neve


2. lépés

Lépjünk be a mappába.

CMD-ben cd mappa


3. lépés

Ha sikeresen beléptünk a mappába "indítsuk" el a git-et.

CMD-ben Git init


Initialized empty Git repository in /Users/felhasználók/mappa/.git


Létrehoztunk egy Git repository-t
